# 意义编织 (Meaning Weaver)

一个帮你从日常生活碎片中发现隐藏模式和自我洞察的 AI 日记工具。

**不是日记 App，是一面帮你看见自己的镜子。**

## 它能做什么

用户通过文字、语音、照片随手记录生活碎片——不需要完整的日记，只需要捕捉瞬间的想法、感受、事件。每周，AI 会分析这些碎片，生成一份「生活线索报告」，帮你看见自己的关注点、情绪模式、价值倾向和成长轨迹。

### 功能

- **多模态碎片输入** — 文字、语音转文字、照片，什么方便用什么
- **AI 周报** — 从一周碎片中提炼主题、模式、洞察
- **历史周报浏览** — 回顾过去每周的报告
- **本地数据存储** — 所有个人数据保存在设备本地，不上传服务器

## 安装

### Android

前往 [Releases](https://github.com/simonlei/meaning-weaver/releases) 下载最新的 APK 安装包。

### 本地开发

```bash
# 安装依赖
npm install

# 启动 Metro bundler
npm start

# 启动代理服务器（语音识别和 Web 开发需要）
node proxy-server.js

# 运行到设备/模拟器
npm run android
npm run ios
npm run web
```

## 配置

首次使用需要在「设置」页面配置：

1. **腾讯混元 API Key** — 用于 AI 周报生成和照片描述
2. **腾讯云 ASR 凭证**（可选） — SecretId + SecretKey，用于语音转文字

## 技术栈

| 层 | 技术 |
|---|------|
| 框架 | React Native + Expo |
| 语言 | TypeScript |
| 本地数据库 | expo-sqlite |
| AI 模型 | 腾讯混元（文本 + 视觉） |
| 语音识别 | 腾讯云 ASR |
| 状态管理 | @tanstack/react-query |
| 数据校验 | Zod |

## 设计理念

- **反效率** — 不催促多记录，不搞打卡机制
- **不评判** — AI 不说「你应该怎样」，只说「我注意到你在关注这些」
- **本地优先** — 隐私数据不离开你的设备
- **低摩擦** — 打开即可记录，无需分类打标签
